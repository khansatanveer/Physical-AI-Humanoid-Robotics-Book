# Further Reading - Isaac Sim

## Official Documentation

- [Isaac Sim Documentation](https://docs.omniverse.nvidia.com/isaacsim/latest/overview.html) - Complete official documentation for Isaac Sim, including installation, API reference, and tutorials
- [NVIDIA Omniverse Documentation](https://docs.omniverse.nvidia.com/) - Documentation for the underlying Omniverse platform that powers Isaac Sim
- [USD (Universal Scene Description) Specification](https://graphics.pixar.com/usd/release/docs/index.html) - Official specification for the scene description format used by Isaac Sim

## Tutorials and Examples

- [Isaac Sim Tutorials](https://docs.omniverse.nvidia.com/isaacsim/latest/tutorial.html) - Step-by-step tutorials for getting started with Isaac Sim
- [Isaac ROS Documentation](https://nvidia-isaac-ros.github.io/repositories_and_packages.html) - Documentation for Isaac ROS packages that integrate with Isaac Sim
- [Isaac Gym Examples](https://github.com/NVIDIA-Omniverse/IsaacGymEnvs) - Reinforcement learning environments using Isaac Sim

## Research Papers

- ["Isaac Gym: High Performance GPU Based Reinforcement Learning for Physics Simulation"](https://arxiv.org/abs/2108.10470) - Research paper on high-performance RL in physics simulation
- ["Physically Accurate Real-time Rendering of Cloth using Neural Networks"](https://research.nvidia.com/publication/2021-08_Physically-Accurate-Real-time) - Research on photorealistic rendering in simulation

## Community and Support

- [NVIDIA Developer Forums](https://forums.developer.nvidia.com/c/omniverse/isaac-sim/) - Community forums for Isaac Sim users and developers
- [Isaac ROS GitHub](https://github.com/NVIDIA-ISAAC-ROS) - GitHub repositories for Isaac ROS packages
- [NVIDIA Omniverse Blog](https://developer.nvidia.com/blog/tag/omniverse/) - Blog posts about Omniverse and Isaac technologies

## Related Technologies

- [ROS 2 Documentation](https://docs.ros.org/en/humble/) - Documentation for ROS 2, which integrates with Isaac Sim
- [Navigation2 (Nav2) Documentation](https://navigation.ros.org/) - Documentation for the navigation stack mentioned in this book
- [Gazebo Simulation](http://gazebosim.org/) - Alternative physics simulation platform for robotics
