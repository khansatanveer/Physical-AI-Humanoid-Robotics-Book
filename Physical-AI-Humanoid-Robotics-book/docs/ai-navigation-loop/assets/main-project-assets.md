# Main Project Assets: Complete AI-Driven Navigation Loop

## Visual Assets Overview

This directory contains all visual demonstrations, 3D models, and multimedia assets for the Complete AI-Driven Navigation Loop project.

## 3D Models

### Robot Models
- `humanoid_robot.urdf` - Main humanoid robot model for simulation
- `humanoid_robot_collision.stl` - Collision geometry for physics simulation
- `humanoid_robot_visual.dae` - Visual mesh for rendering

### Environment Models
- `navigation_environment.urdf` - Complete navigation environment
- `obstacle_objects.urdf` - Various obstacle objects for testing
- `testing_arena.stl` - Navigation testing arena model
