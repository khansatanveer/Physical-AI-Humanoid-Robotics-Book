# Isaac Example Assets

This directory contains various assets used in the Isaac examples:

## 3D Models
- `3d-models/` - 3D models for robots, environments, and objects
- File formats: `.usd`, `.usda`, `.obj`, `.fbx`

## Diagrams
- `diagrams/` - Technical diagrams and illustrations
- File formats: `.png`, `.svg`, `.jpg`

## Visual Demonstrations
- `demos/` - GIFs and videos showing example outputs
- File formats: `.gif`, `.mp4`, `.mov`

## Textures and Materials
- `textures/` - Texture files and material definitions
- File formats: `.png`, `.jpg`, `.dds`
